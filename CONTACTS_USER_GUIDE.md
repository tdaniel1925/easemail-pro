# Contacts User Guide

## Quick Start

### Accessing Contacts
Navigate to the **Contacts** section from the sidebar to view and manage all your contacts.

## Features

### 1. Adding Contacts

#### Manual Entry
1. Click the **"Add Contact"** button in the top-right
2. Fill in the contact information:
   - **Required**: First Name, Email
   - **Optional**: Last Name, Phone, Company, Job Title, Social links, Location, Tags, Notes
3. Click **"Create Contact"**

#### From Email
1. Open any email
2. In the right sidebar, click **"Add to Contacts"**
3. Contact form auto-fills with sender's information
4. AI automatically enriches the contact data
5. Review and save

#### CSV Import
1. Click **"Import"** button
2. Download the template (optional)
3. Upload your CSV file
4. Review import results

### 3. Searching & Filtering

#### Search
Type in the search box to find contacts by:
- Name
- Email
- Company
- Job Title

#### Filters
- **Tag Filter**: Select a tag from the dropdown to see only contacts with that tag
- **Sort Options**:
  - By Name (A-Z)
  - By Recent (most recent first)
  - By Email Count (most active)
  - By Company (A-Z)

#### Send Email

### 6. Import/Export

#### Export Contacts
1. Click **"Export"** button
2. CSV file downloads automatically
3. Contains all contact information

   Location, Website, LinkedIn, Twitter, Tags, Notes

### 7. Contact Statistics

Each contact displays:
- **Email Count**: Total email threads
- **Last Contact**: Time since last interaction
- **In Contacts**: How long they've been saved
- **Phone Status**: Whether phone number is available

### 8. AI Enrichment

When creating a contact from an email, AI automatically:
- Extracts company name from email signature
- Identifies job title
- Finds LinkedIn profile
- Determines location
- Discovers phone number (if available)
- Suggests tags

You can edit any AI-suggested information before saving.

## Troubleshooting

### Contact Not Saving
- Check that email address is valid
- Ensure first name is provided
- Check for duplicate email (can't have same email twice)

### Import Errors
- Verify CSV format matches template
- Check that email column exists and has valid emails
- Remove duplicate emails from import file
- Ensure no special characters in headers

### AI Enrichment Not Working
- AI enrichment works only when creating from emails
- Requires email content to extract information
- May not find all information for every contact
- You can manually add any missing fields

### Missing Contact Information
- Some fields are optional (company, phone, etc.)
- AI may not find all information
- Edit contact to add manually

## FAQ

**Q: Can I import contacts from Gmail/Outlook?**
A: Export contacts as CSV from your email provider, then import here.

**Q: How do I avoid duplicates?**
A: System automatically prevents duplicate emails. Import will skip existing contacts.

**Q: Can I bulk delete contacts?**
A: Currently one at a time. Bulk operations coming soon.

**Q: Are contacts synced across devices?**
A: Yes, contacts are stored in your account and sync automatically.

**Q: Can I merge duplicate contacts?**
A: Not yet, but you can delete one and manually update the other.

**Q: How many contacts can I have?**
A: No limit! Store as many as you need.

**Q: Can I export specific contacts only?**
A: Use filters/search first, then export. Only visible contacts are exported.
